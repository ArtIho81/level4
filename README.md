## Description

This API is local analog of https://swapi.dev

## Installation

```bash
$ npm install
```

# Running the app

Note: Local PostgreSQL server must be running and empty database 'level-4' has created on it.   

# Running migrations
```bash
$ npm run migration:run
```
# Seeding database from swapi.dev
```bash
$ npm run seed
```

## Starting APP
```bash
$ npm run start
```
